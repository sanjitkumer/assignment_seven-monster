1. What is difference between local and session storage?

Ans: This read-only interface property provides access to the Document’s local storage object, the stored data is stored across browser sessions. Similar to sessionStorage, except that localStorage data gets cleared when the page session ends – that is when the page is closed. It is cleared when the last “private” tab of a browser is closed (localStorage data for a document loaded in a private browsing or incognito session).


2. What is the difference between Gobal scope and block scope?

Ans: Global scope variables can use within the program anyware. Global variables are declaring outside any functions. Local scope variable value is only for that particular scope (function). Local variables are declaring inside that particular function.
